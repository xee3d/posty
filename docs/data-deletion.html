Posty 사용자 데이터 삭제 안내

최종 업데이트: 2025년 7월 22일

Posty는 사용자의 개인정보 자기결정권을 존중하며, 언제든지 본인의 데이터 삭제를 요청할 수 있습니다.

데이터 삭제 요청 방법

방법 1: 앱 내에서 직접 삭제

  1. Posty 앱을 실행합니다
  2. 하단 메뉴에서 마이페이지를 탭합니다
  3. 설정 → 계정 관리로 이동합니다
  4. 계정 삭제 버튼을 탭합니다
  5. 안내에 따라 삭제를 확인합니다

방법 2: 이메일로 요청

  1. 아래 이메일 주소로 삭제 요청을 보내주세요
  2. 이메일: privacy@posty.app
  3. 제목: [데이터 삭제 요청] 사용자명
  4. 본문에 포함할 내용:
    • 가입 시 사용한 이메일 주소
    • 소셜 로그인 종류 (Facebook, Google, 카카오, 네이버, Apple)
    • 삭제 요청 사유 (선택사항)

삭제되는 데이터

데이터 삭제 요청 시 다음의 정보가 완전히 삭제됩니다:

  • 프로필 정보 (이름, 이메일, 프로필 사진)
  • 게시물 및 댓글
  • 좋아요 및 팔로우 정보
  • 앱 사용 기록
  • 소셜 로그인 연동 정보
⚠️ 주의사항
  • 삭제된 데이터는 복구할 수 없습니다
  • 일부 데이터는 법적 의무 준수를 위해 일정 기간 보관될 수 있습니다
  • 다른 사용자가 작성한 콘텐츠에 포함된 정보는 삭제되지 않을 수 있습니다

처리 기간

  • 앱 내 삭제: 즉시 처리
  • 이메일 요청: 접수 후 7일 이내
  • 처리 완료 시 등록된 이메일로 안내 메일을 발송합니다

소셜 로그인 데이터

Posty에서 계정을 삭제해도 소셜 로그인 제공자(Facebook, Google 등)에 저장된 데이터는 삭제되지 않습니다. 각 플랫폼에서 Posty 앱 연동을 해제하려면:

  • Facebook: 설정 → 앱 및 웹사이트 → Posty 제거
  • Google: 계정 설정 → 보안 → 제3자 앱 액세스 → Posty 액세스 권한 삭제
  • 카카오: 카카오계정 → 연결된 서비스 관리 → Posty 연결 끊기
  • 네이버: 내정보 → 보안설정 → 외부 사이트 연결 → Posty 삭제
  • Apple: 설정 → Apple ID → 로그인 및 보안 → Apple로 로그인 → Posty 제거
