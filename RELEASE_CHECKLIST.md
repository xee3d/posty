## 🔍 프로젝트 개요

### **앱 정보**
- **앱 이름**: Posty (포스티)
- **버전**: 1.0.0
- **플랫폼**: React Native 0.74.5
- **주요 기능**: AI 기반 SNS 콘텐츠 생성 앱

### **기술 스택**
- **Frontend**: React Native, TypeScript, Redux Toolkit
- **Backend**: Firebase (Auth, Firestore, Analytics)
- **AI**: OpenAI API (GPT-4)
- **광고**: Google AdMob
- **결제**: React Native IAP
