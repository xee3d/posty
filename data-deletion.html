Posty 사용자 데이터 삭제 안내

최종 업데이트: 2025년 8월 28일

Posty는 사용자의 개인정보 자기결정권을 존중하며, 언제든지 본인의 데이터 삭제를 요청할 수 있습니다. GDPR 및 개인정보보호법에 따라 사용자의 '잊힐 권리'를 보장합니다.

데이터 삭제 요청 방법

🚀 방법 1: 앱 내에서 즉시 삭제 (권장)

✅ 즉시 처리: 앱 내에서 바로 계정 삭제 시 모든 데이터가 즉시 삭제됩니다.
  1. 1Posty 앱을 실행합니다
  2. 2하단 메뉴에서 마이페이지를 탭합니다
  3. 3설정 → 계정 관리로 이동합니다
  4. 4계정 삭제 버튼을 탭합니다
  5. 5삭제 확인 메시지를 읽고 "영구 삭제"를 확인합니다
  6. 6삭제가 완료되면 자동으로 로그아웃됩니다

📧 방법 2: 이메일로 요청

앱에 접근할 수 없거나 추가 지원이 필요한 경우 이메일로 요청하실 수 있습니다.

받는 사람: getposty@gmail.com

감사합니다.

  • 가입 시 사용한 이메일 주소
  • 소셜 로그인 종류 (Facebook, Google, 카카오, 네이버, Apple)
  • 본인 확인 (계정 소유자임을 확인하는 문구)
  • 삭제 요청 사유 (선택사항)

삭제되는 데이터 목록

데이터 삭제 요청 시 다음의 정보가 완전히 그리고 영구적으로 삭제됩니다:

🔍 개인 정보

  • 프로필 정보 (이름, 이메일, 프로필 사진, 닉네임)
  • 소셜 로그인 연동 정보
  • 구독 정보 및 결제 이력
  • 앱 설정 및 사용자 선호도

📝 콘텐츠 및 사용 기록

  • 생성한 모든 AI 콘텐츠
  • 저장한 텍스트 및 즐겨찾기
  • 토큰 사용 이력
  • 앱 사용 통계 및 분석 데이터
  • 접속 로그 및 기기 정보

🎯 광고 관련 데이터

  • 광고 식별자(ADID) 정보
  • 맞춤형 광고 프로필
  • 광고 상호작용 기록

⚠️ 중요 주의사항

  • 복구 불가능: 삭제된 데이터는 어떠한 방법으로도 복구할 수 없습니다
  • 법적 보관: 일부 데이터는 관련 법령(전자상거래법, 통신비밀보호법 등)에 따라 일정 기간 보관될 수 있습니다
  • 백업 시스템: 백업 시스템에서의 완전 삭제는 최대 30일이 소요될 수 있습니다
  • 익명화된 통계: 개인을 식별할 수 없는 통계 데이터는 서비스 개선 목적으로 보관될 수 있습니다

처리 기간 및 절차

삭제 방법 처리 시간 확인 방법
앱 내 삭제 즉시 처리 앱에서 자동 로그아웃
이메일 요청 접수 후 7일 이내 완료 안내 이메일 발송

📋 처리 절차

  1. 요청 접수: 이메일 요청 시 24시간 내 접수 확인
  2. 본인 확인: 계정 소유자 확인 절차 진행
  3. 데이터 삭제: 모든 시스템에서 데이터 완전 삭제
  4. 완료 통보: 삭제 완료 이메일 발송

소셜 로그인 연동 해제

Posty에서 계정을 삭제해도 소셜 로그인 제공자에 저장된 데이터는 삭제되지 않습니다. 각 플랫폼에서 Posty 앱 연동을 해제하려면:

🔗 소셜 플랫폼별 연동 해제 방법

  • 🔵 Facebook: 설정 → 앱 및 웹사이트 → Posty 제거
  • 🔴 Google: 계정 설정 → 보안 → 제3자 앱 액세스 → Posty 액세스 권한 삭제
  • 🟡 카카오: 카카오계정 → 연결된 서비스 관리 → Posty 연결 끊기
  • 🟢 네이버: 내 정보 → 보안설정 → 외부 사이트 연결 → Posty 삭제
  • ⚫ Apple: 설정 → Apple ID → 로그인 및 보안 → Apple로 로그인 → Posty 제거
